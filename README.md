# ProthSync

## 이 프로젝트를 시작하게 된 계기

- 처음에는 치과랑 치과 기공소만 연결을 하려고 했다.
- 그래서 배달의 민족 처럼 치과가 이 웹 혹은 앱을 키면 근처의 치과 기공소가 뜨고 그 치과 기공소의 포트폴리오를 확인하는 그런 방향을 생각했었다.
- 그런데 어제 대학교 친한 후배랑 대화를 해봤는데 학생들도 약간 취업 준비를 위해서 학생들 포트폴리오를 올리고 치과 기공소에서 확인하고 방학 때 실습이라던지 아니면 추후에 졸업생들은 취업까지 알선 될 수 있도록 약간 좋은 실력을 가진 후배들이 좋은 기공소에서 먼저 컨택해서 할 수 있도록 그런 방향을 추가로 기획 하는게 좋을 것 같아서 이 프로젝트를 진행

## 생각하고 있는 개발 방향

- User의 종류는 치과, 치과 기공 소장, 치과 기공사, 학생 이렇게 된다.
- 학생은 자기의 포트폴리오를 올리는 공간이 있다.
  - 약간 블로그나, 인스타 처럼
- 치과 기공사와 치과 기공 소장 또한 자신들의 포트폴리오를 올리는 공간이 있다.

- 학생이 올린 포트폴리오는 치과 기공소장이 볼 수 있다.
- 치과 기공사가 올린 포트폴리오는 치과 기공소장이 볼 수 있다.
- 치과 기공소장이 올린 포트폴리오는 치과에서 확인 할 수 있다.
